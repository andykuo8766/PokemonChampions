# 优先度

來源：神奇宝贝百科
原始網址：https://wiki.52poke.com/wiki/%E4%BC%98%E5%85%88%E5%BA%A6

優先度（日文︰優先度，英文︰Priority）是決定對戰中雙方使用招式先後的其中一個因素。

## 概述

每個招式都有自己的優先度。當對戰中要判定兩隻寶可夢誰先進行攻擊時，會先判定優先度。當一方使用的招式優先度較高時，這隻寶可夢無論速度如何，都會比其他寶可夢更快地使出招式。相反，當一方的招式優先度較低時，就算牠速度再高也會比對手慢出。只有在雙方使用的招式優先度相同時，才要求判定速度。優先度的範圍為 -7 至 +5，通常的招式優先度是 0，先制招式的優先度為正，戲法空間等一些特殊招式的優先度為負。

- 招式的優先度不受戲法空間影響，戲法空間只影響相同優先度下根據速度出招的順序。
- 地面上的寶可夢在精神場地上、有鮮艷之軀、女王的威嚴、尾甲等特性在場時，或對方場地為快速防守狀態時，使用優先度大於 0 的招式將會失敗。
- 如果有目標在本回合內準備替換下場，追打的優先度比以上所有招式都高，但在使用追打前，寶可夢已經瀕死，則優先級仍依照原先招式的順序。
- 道具先制之爪、釋陀果、後攻之尾、飽腹薰香與特性慢出、速擊、菌絲之力可以改變招式使用順序，但不影響招式本身的優先度。
- 招式您先請、延後、號令及特性舞者可以改變當回合招式使用順序，但不影響招式優先度。
- 《傳說 阿爾宙斯》中不使用優先度，而是用行動順序判定。《傳說 Z-A》中不存在優先度。

## 優先度變更

- 吹飛、吼叫：0 → -1（第二世代）
- 吹飛、吼叫：-1 → -6（第三世代）
- 忍耐：0 → 1（第四世代）
- 擊掌奇襲：1 → 3（第五世代）
- 神速：1 → 2（第五世代）
- 守住、看穿、挺住：3 → 4（第五世代）
- 報復：0 → -4（第五世代）
- 憤怒粉、看我嘛：3 → 2（第六世代）
- 魔法空間、奇妙空間：-7 → 0（第六世代）
- 交換場地：1 → 2（第七世代）
- 瞬間移動：0 → -6（第七世代）

## 可以變更優先度的特性

- 惡作劇之心：變化招式的優先度 +1
- 疾風之翼：HP 全滿時，飛行屬性招式的優先度 +1
- 先行治療：治癒類招式的優先度 +3

## 細節

- 真氣拳的優先度高於雪崩、報復、雙倍奉還、龍尾與巴投。這五個招式也是少數無法打斷真氣拳的攻擊招式。
